# OpenClaw 展示网站

展示 OpenClaw 使用案例和提供合作方案的企业官网。

## 技术栈

- **后端**: Python + FastAPI
- **前端**: HTML + TailwindCSS + Alpine.js
- **数据库**: SQLite (开发) / PostgreSQL (生产)

## 项目结构

```
openclaw-showcase/
├── backend/           # Python FastAPI 后端
│   ├── app/
│   ├── data/          # 数据文件
│   └── requirements.txt
├── frontend/          # 前端静态文件
│   ├── index.html
│   ├── css/
│   └── js/
└── README.md
```

## 内容模块

1. **OpenClaw 入门** - 官方教程 + 使用经验
2. **真实案例** - 我们的场景应用
3. **Skills 介绍** - 官方 + 自建
4. **合作方案** - 5种服务类型

## 合作方案

1. 个人咨询服务
2. 企业咨询服务
3. 部署软件服务
4. Skill 销售
5. Skill 定制化研发 / OpenClaw 定制改造
